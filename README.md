### Typography

| Role | Font | Weight |
|---|---|---|
| Display / Headlines | Bebas Neue | 400 |
| Body / Descriptions | Manrope | 300, 400, 500, 600, 700, 800 |
| Monospace / Labels | JetBrains Mono | 300, 400 |

### Animation System

- Custom cursor (mint dot + outer ring, hover-reactive)
- Preloader with letter-by-letter reveal and progress counter
- Scroll-triggered reveal animations (translateY + opacity)
- Three.js 3D scenes (unique per page — no reuse)
- Magnetic button physics on hover
- 3D card tilt with `perspective()` transform
- Staggered entrance animations with `animation-delay`
